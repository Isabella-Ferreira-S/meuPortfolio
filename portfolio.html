Projeto 2: Calculadora Geométrica

Tecnologias Utilizadas:

  • JavaScript: Utilizado para criar uma calculadora geométrica interativa. O JavaScript permite realizar cálculos baseados nas entradas do usuário e exibir os resultados através de diálogos de prompt e alert. O código permite calcular áreas de diferentes formas geométricas, incluindo triângulo, retângulo, quadrado, trapézio e círculo.

Descrição: Este projeto consiste em uma calculadora que permite ao usuário calcular a área de várias formas geométricas. O programa apresenta um menu interativo para selecionar a forma desejada e, em seguida, solicita ao usuário as dimensões necessárias para realizar o cálculo. O uso do JavaScript possibilita a interação direta com o usuário e a execução de cálculos matemáticos com base nas entradas fornecidas.

GitHub - Calculadora Geométrica

Projeto 3: Jogo da Forca em Python

Tecnologias Utilizadas:

  • Python: Utilizado para desenvolver um jogo da forca. O Python foi escolhido pela sua simplicidade e clareza na manipulação de dados e interação com o usuário. O código permite a leitura de palavras a partir de um arquivo, a execução do jogo com base em tentativas do usuário, e o registro da pontuação em um arquivo de texto.

Descrição: Este projeto é um jogo da forca desenvolvido em Python. O jogo sorteia uma palavra de uma lista carregada a partir de um arquivo de texto e permite que o usuário tente adivinhar a palavra letra por letra. O código inclui funções para desenhar o estado do jogo e registrar a pontuação final em um arquivo. Este projeto destaca a utilização de Python para manipulação de arquivos, controle de fluxo e interação com o usuário.

GitHub - Jogo da Forca em Python
